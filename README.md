darkhn
======

Dark theme for HackerNews

# Install
1. Grab a copy of [Stylish](https://chrome.google.com/webstore/detail/stylish/fjnbnpbmkenffdnngjfgmeleoegfcffe).
2. Make a new theme, copy paste in the CSS.
